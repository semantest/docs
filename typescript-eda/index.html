TypeScript-EDA Framework

Note: TypeScript-EDA has evolved into the Semantest Framework. This page provides information about the underlying event-driven architecture that powers Semantest.

About TypeScript-EDA

TypeScript-EDA is the core event-driven architecture framework that provides the foundation for the Semantest testing platform. It implements Domain-Driven Design (DDD) patterns with event sourcing capabilities.

Key Features

  • Event-Driven Architecture: Built on proven DDD and EDA patterns
  • @listen Decorators: Elegant event handling with TypeScript decorators
  • Domain Entities: Rich domain models with value objects
  • Hexagonal Architecture: Clean separation of concerns
  • Test-Driven Development: Comprehensive testing support

Architecture Overview

TypeScript-EDA (Core Framework)
    │
    ├── Domain Layer
    │   ├── Entities
    │   ├── Value Objects
    │   └── Domain Events
    │
    ├── Application Layer
    │   ├── Command Handlers
    │   ├── Event Handlers (@listen)
    │   └── Query Handlers
    │
    └── Infrastructure Layer
        ├── Repositories
        ├── Message Bus
        └── External Services
